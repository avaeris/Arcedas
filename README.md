## Features

- Small string utility helpers (slugify, sanitize)
- Pytest-based tests
- GitHub Actions CI for running tests

## Getting started

Clone the repo and run tests locally:

```bash
git clone git@github.com:avaeris/Arcedas.git
cd Arcedas
python -m pip install -r requirements.txt
python -m pytest -q
```
